quiver2html
===========

A utility to convert a Quiver notebook or note to HTML.

## Installation

Install quiver2html using npm:

```bash
$ [sudo] npm install -g quiver2html
```

## Usage

```bash
quiver2html test.qvnotebook -o output_dir
quiver2html test.qvnote -o output_dir
```
